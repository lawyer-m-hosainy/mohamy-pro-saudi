# 🏛️ محامي برو — Mohamy Pro
## المنصة القانونية المتكاملة لإدارة مكاتب المحاماة في المملكة العربية السعودية

---

> **ملخص تنفيذي:** محامي برو هي أول منصة سحابية عربية متكاملة من نوعها، صُمّمت خصيصاً لتلبية احتياجات مكاتب وشركات المحاماة في المملكة العربية السعودية. تجمع المنصة بين إدارة القضايا، الشؤون المالية، الذكاء الاصطناعي، والامتثال القانوني السعودي في نظام واحد موحّد — مما يُغني عن الحاجة لعدة أنظمة منفصلة.

## 🧩 الوحدات والمميزات التفصيلية (أكثر من 30 وحدة)

### 🎯 الوحدات الأساسية (Core Modules)

#### 1. لوحة القيادة (Dashboard)
- **نظرة عامة فورية** على أداء المكتب: عدد القضايا، العملاء، المهام، الإيرادات
- **رسوم بيانية تفاعلية** لتحليل الإيرادات وتوزيع القضايا
- **صياغة ذكية بالذكاء الاصطناعي** — إنشاء لوائح دعوى ومذكرات جوابية ولوائح اعتراضية وعقود بضغطة زر
- **تنبيهات نظام ذكية** — مواعيد جلسات، تحديثات ناجز، عملاء جدد

#### 2. إدارة العملاء (Clients / CRM)
- قاعدة بيانات شاملة لجميع العملاء (أفراد ومنشآت)
- **نظام هوية سعودي**: الهوية الوطنية، السجل التجاري، الرقم الضريبي (15 رقم — ZATCA)
- خرائط العلاقات المتقدمة: الشركة الأم، الشركات التابعة، المساهمون، أعضاء مجلس الإدارة
- رقم الهاتف بتنسيق سعودي (`+9665XXXXXXXX`)
- البحث والفلترة المتقدمة مع ترقيم الصفحات

#### 3. إدارة القضايا (Cases)
- إدارة كاملة لدورة حياة القضية من الاستلام إلى الإغلاق
- **5 مراحل عمل** (Workflow): استلام → لوائح → جلسات → حكم → إغلاق
- دعم كافة محاكم المملكة: العليا، الاستئناف، العامة، الجزائية، التجارية، العمالية، الأحوال الشخصية، ديوان المظالم
- **ربط مع منصة ناجز** العدلية
- رفع المذكرات والوثائق ومرفقاتها
- تتبع الجلسات والمواعيد النهائية
- ملخص مالي لكل قضية (مصروفات، جلسات، مهام)

#### 4. تقويم الجلسات (Calendar)
- عرض شهري/أسبوعي لجميع مواعيد الجلسات
- تنبيهات استباقية قبل المواعيد

#### 5. إدارة المهام (Tasks)
- إنشاء مهام مرتبطة بالقضايا
- تعيين المهام لأعضاء الفريق
- تصنيف حسب الأولوية (عالية/متوسطة/عادية)
- متابعة الإنجاز في الوقت الحقيقي

---

### 💰 الوحدات المالية (Financial Modules)

#### 6. المالية والضريبة (Finance)
- **فواتير إلكترونية متوافقة مع ZATCA** (ضريبة القيمة المضافة 15%)
- حالات الفاتورة: مدفوعة / غير مدفوعة / مسودة
- تقارير مالية شاملة

#### 7. المصروفات (Expenses)
- تسجيل مصروفات كل قضية (رسوم قضائية، أتعاب خبراء، تنقلات)
- حالات الدفع: تم السداد / معلق / مسترد من العميل

#### 8. حسابات الأمانة (Trust Accounting)
- حسابات أمانات العملاء والمبالغ المقبوضة
- أنواع: أمانة / مقدم أتعاب / مبلغ تنفيذ
- تتبع حالة كل مبلغ

#### 9. تتبع الوقت (Time Tracking)
- تسجيل ساعات العمل لكل محامي على كل قضية
- تصنيف: مفوتر / غير مفوتر
- أساس لاحتساب أتعاب الساعة

#### 10. تحصيل الديون (Collections)
- تتبع المستحقات والمبالغ المتأخرة
- إجراءات التحصيل: إصدار مطالبة / إنذار قانوني / جدولة سداد / تسوية
- خطط سداد بالأقساط

### ⚖️ الوحدات القانونية المتخصصة (Legal Operations)

#### 11. إدارة التنفيذ (Enforcement)
- متابعة قضايا التنفيذ وأوامره
- تتبع أصول المدين (حسابات بنكية، عقارات، مركبات)
- أوامر التنفيذ: إشعار 46، أمر حجز، منع سفر

#### 12. فحص التعارض (Conflict Check)
- فحص تضارب المصالح التلقائي قبل قبول أي قضية
- كشف التعارض: مباشر / غير مباشر / تنازل
- مستوى الخطورة: عالي / متوسط / منخفض

#### 13. بوابة الاستشارات (Advisory Desk)
- إدارة طلبات الاستشارات القانونية
- سير عمل: جديد → قيد المراجعة → مسودة رأي → اعتماد شريك → مغلق
- **مستوى اتفاقية الخدمة (SLA)** لكل طلب

#### 14. المسارات المتخصصة (Specialized Tracks)
- مسارات عمل مخصصة للقضايا العمالية والجزائية
- قوائم تدقيق (Checklists) إلزامية لكل مرحلة
- قوالب وثائق جاهزة لكل مسار

#### 15. إدارة العقود (CLM - Contract Lifecycle Management)
- دورة حياة العقد الكاملة: طلب → تفاوض → مراجعة → اعتماد → توقيع → متابعة التزامات → تجديد/إنهاء
- **تعدد الإصدارات** مع تتبع التغييرات
- سلسلة اعتماد متعددة المستويات
- متابعة الالتزامات التعاقدية

#### 16. صانع العقود الذكي (Smart Contract Builder)
- قوالب عقود جاهزة: تجاري، عمالي، عقاري، أحوال شخصية
- تخصيص القوالب حسب الحاجة

#### 17. الملكية الفكرية (IP Operations)
- تسجيل العلامات التجارية وبراءات الاختراع وحقوق المؤلف
- متابعة الإيداعات والتجديدات
- إدارة الاعتراضات
- إجراءات حماية الملكية الفكرية

#### البنية التحتية المبنية (Production-Ready):

| المكون | الحالة | التفاصيل |
|---|---|---|
| **خادم Backend** | ✅ مبني | Express.js Server مع 3 نقاط API مخصصة |
| **النموذج** | ✅ مُعدّ | Google Gemini 2.5 Flash (أحدث وأسرع نموذج) |

| **التفعيل** | 🔑 مفتاح API فقط | إدخال `GEMINI_API_KEY` في إعدادات Render = تشغيل فوري |

#### 18. المحلل القانوني الذكي (AI Document Analyzer) — ✅ مبني
- تحليل اللوائح والأحكام والمذكرات القانونية
- استخراج الدفوع والثغرات والمخاطر وفق **الأنظمة السعودية**
- **API Endpoint**: `POST /api/ai/analyze`
- حد أقصى للنص: 50,000 حرف
- واجهة: إدخال نص + عرض نتائج التحليل مع خيارات النسخ والتنزيل

#### 19. الصياغة الذكية (AI Legal Drafting) — ✅ مبني
- صياغة 5 أنواع من الوثائق: لائحة دعوى، مذكرة جوابية، لائحة اعتراضية، عقد تأسيس شركة، اتفاقية عدم إفصاح
- **API Endpoint**: `POST /api/ai/draft`
- حد أقصى للوقائع: 10,000 حرف
- متاح مباشرة من لوحة القيادة الرئيسية

#### 20. المساعد القانوني الذكي (AI Chat Assistant) — ✅ مبني
- مساعد دردشة ذكي **متاح في كل صفحة** من المنصة (زر عائم)
- متخصص في **الأنظمة السعودية** مع تعليمات نظام مخصصة
- يحتفظ بسياق المحادثة (Chat History)
- **API Endpoint**: `POST /api/ai/legal-assistant`
- حد أقصى للرسالة: 5,000 حرف
- قابل للتكبير (Maximize) لتجربة مريحة

هذه الوحدات تمثل **ميزة تنافسية استثنائية** في السوق السعودي:

## 🏗️ البنية التقنية

| المكون | التقنية |
|---|---|
| **الواجهة الأمامية** | React 19 + TypeScript + Vite |
| **التصميم** | Tailwind CSS + Radix UI + Framer Motion |
| **قاعدة البيانات** | Firebase Firestore (NoSQL, Real-time) |
| **المصادقة** | Firebase Authentication |
| **التخزين** | Firebase Storage |
| **الذكاء الاصطناعي** | Google Gemini AI |
| **الاستضافة** | Render (Cloud) |
| **الرسوم البيانية** | Recharts |
| **البنية** | Multi-Tenant SaaS Architecture |
| **الأمان** | RBAC (Role-Based Access Control) + Error Boundaries + Audit Logging |

### مميزات تقنية بارزة:
- ✅ **بنية Multi-Tenant**: كل مكتب محاماة له بيئة مستقلة ومعزولة
- ✅ **التحميل الكسول (Lazy Loading)**: كل صفحة تُحمّل عند الطلب لأداء فائق
- ✅ **نظام صلاحيات (RBAC)**: 6 أدوار مختلفة بصلاحيات محددة
- ✅ **واجهة RTL كاملة**: دعم اللغة العربية بالكامل
- ✅ **الوضع الداكن/الفاتح**: تجربة مريحة للمستخدم
- ✅ **متجاوبة (Responsive)**: تعمل على الحاسب والجوال والجهاز اللوحي
- ✅ **Health Checks**: مراقبة صحة النظام تلقائياً كل دقيقة
- ✅ **Error Boundaries**: حماية من الانهيار الكامل للتطبيق

لا يوجد منافس سعودي متكامل يغطي كل احتياجات مكتب المحاماة. معظم المكاتب تستخدم Excel أو أنظمة أجنبية غير ملائمة.

### 2. 🟢 منتج جاهز وليس فكرة
المنصة **مبنية وتعمل فعلاً** بأكثر من 30 وحدة — وليست مجرد عرض تقديمي.

### 3. 🟡 نموذج SaaS متكرر الإيرادات
اشتراكات شهرية/سنوية = إيرادات متكررة ومتوقعة (Monthly Recurring Revenue).

### 4. 🔴 توقيت مثالي
- رؤية 2030 تدفع نحو الرقمنة
- وزارة العدل تشجع التحول الرقمي
- منصة ناجز تفرض التعامل الإلكتروني
- نظام المحاماة الجديد يفرض معايير احترافية

### 5. 🟣 إمكانية التوسع
- التوسع لباقي دول الخليج (الإمارات، الكويت، البحرين)
- التوسع للإدارات القانونية في الشركات
