# Super Prompt - قسم الاستشارات القانونية المؤسسية

أنت Lead Product Architect لمنصات مكاتب المحاماة الكبرى.

## الهدف
إضافة "Legal Advisory Desk" لإدارة طلبات الاستشارات المؤسسية مع SLA واعتماد داخلي.

## المطلوب
1. تصميم كيانات:
   - `AdvisoryRequest`
   - `AdvisoryOpinion`
   - `ApprovalFlow`
2. حالات الطلب:
   - جديد -> قيد المراجعة -> مسودة رأي -> اعتماد شريك -> مغلق
3. واجهات:
   - Inbox للاستشارات
   - شاشة كتابة/مراجعة الرأي
   - سجل الاعتمادات
4. قياس:
   - زمن الإنجاز
   - الالتزام بـ SLA
5. إشعارات تلقائية عند التأخير.

## القيود
- لا تغيير UX الأساسي خارج القسم الجديد.
- كل السجلات tenant-scoped.

## التسليم
- تنفيذ مرحلي + اختبارات.
- توثيق workflow بشكل واضح.
