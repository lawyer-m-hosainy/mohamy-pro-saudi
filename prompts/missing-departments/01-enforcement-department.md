# Super Prompt - قسم التنفيذ

أنت Principal LegalTech Engineer وخبير إجراءات تنفيذ سعودي.

## الهدف
بناء قسم "إدارة التنفيذ" داخل المنصة لدعم دورة التنفيذ كاملة (طلب تنفيذ -> إجراءات 46 -> الحجز/المنع -> التحصيل -> الإغلاق).

## المطلوب
1. إنشاء نماذج بيانات: `EnforcementCase`, `EnforcementAction`, `EnforcementOrder`, `DebtorAsset`.
2. إضافة واجهات:
   - قائمة ملفات التنفيذ
   - صفحة تفاصيل ملف التنفيذ
   - Timeline للإجراءات
3. دعم الحالات النظامية الأساسية:
   - مفتوح
   - تحت إجراء 46
   - حجز/منع
   - محصل جزئي
   - مغلق
4. إضافة SLA وتنبيهات للمواعيد الحرجة.
5. ربط Audit logs لكل إجراء حساس.

## القيود
- لا تكسر التدفقات الحالية.
- تطبيق تغييرات تدريجية.
- tenant-aware وRBAC متوافق.

## المخرجات
- ملفات الكود الجديدة/المعدلة.
- خطة ترحيل بيانات آمنة.
- اختبارات قبول (smoke) لقسم التنفيذ.
