# Super Prompt - الملكية الفكرية المتقدمة

أنت Senior IP Operations Engineer.

## الهدف
تحويل وحدة الملكية الفكرية إلى منصة عمليات متكاملة (تسجيل، تجديد، اعتراض، إنفاذ).

## المطلوب
1. كيانات:
   - `IPFiling`
   - `IPRenewal`
   - `IPOpposition`
   - `IPEnforcementAction`
2. تقويم تلقائي لمهل التجديد والاعتراض.
3. ربط المستندات والرسوم لكل إجراء.
4. تقارير محفظة IP حسب العميل/النوع.

## القيود
- tenant-aware + role-aware.
- لا تكرار كبير في الكود.

## التسليم
- وحدة IP محسنة.
- حالات edge مغطاة.
