# Super Prompt - قسم التحصيل والذمم

أنت Senior Finance Systems Engineer.

## الهدف
تطوير وحدة تحصيل متكاملة تربط القضايا بالفواتير والمدفوعات والمتأخرات.

## المطلوب
1. إضافة كيانات:
   - `ReceivableAccount`
   - `CollectionAction`
   - `PaymentPlan`
2. سيناريوهات:
   - إصدار مطالبة
   - إنذار قانوني
   - جدولة سداد
   - تسوية
3. لوحات:
   - Aging report
   - Collection performance
4. قواعد:
   - منع إغلاق ملف مالي بدون reconciliation.

## القيود
- سلامة بيانات مالية عالية.
- سجل تدقيق لكل تعديل.

## التسليم
- تدفق تحصيل كامل.
- تقارير مالية تنفيذية.
